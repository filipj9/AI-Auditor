# AI-Auditor (Manifest V3)

An advanced, production-grade Chrome Extension designed to automate complex, multi-section compliance and audit forms for official environmental platforms (e.g., NFOŚiGW). 

The tool elements eliminate repetitive manual data entry by intelligently mapping data structures and automatically filling form fields across dozens of operational sections in milliseconds using structured payloads.

## 🛠️ Key Features
- **Intelligent Field Mapping:** Automatically detects and pairs form elements across 17+ sections and 360+ unique input fields.
- **JSON Data Injection:** Supports direct loading and validation of structured JSON payloads generated by internal AI systems.
- **Manifest V3 Compliant:** Built using modern Google Chrome extension architecture for maximum security, privacy, and performance.
- **Real-time Error Tracking:** Built-in validation logs to monitor missing or incorrectly formatted form inputs.

## ⚙️ Tech Stack
- **AI Core / Orchestrator:** Hermes AI Engine (Custom Agentic Framework for automated compliance mapping and secure data parsing)
- **Architecture:** Chrome Extensions API (Manifest V3)
- **Frontend / Core:** Vanilla JavaScript, HTML5, CSS3
- **Data Format:** JSON for secure state management and automated payloads

## 🚀 Installation & Usage (Developer Mode)
1. Clone this repository or download the source files.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** in the top-left corner.
5. Select the root folder containing the `manifest.json` file.
6. Open the target auditing portal and launch the extension assistant.
